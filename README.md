# Python-Spelling-Corrector-TextBlob
A Python project to automatically correct spelling mistakes using the TextBlob NLP library. Demonstrates basic natural language processing, spelling correction.

This project demonstrates how to correct spelling mistakes in Python using the **TextBlob** Natural Language Processing library.

## What is TextBlob?
TextBlob is a Python library for processing text data. It provides a simple API for NLP tasks like:
- Spelling Correction
- Sentiment Analysis
- Tokenization
- POS Tagging
- Lemmatization
- n-grams
- Classification
